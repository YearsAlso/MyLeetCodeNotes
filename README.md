# MyLeetCodeNotes

## 简介

这个项目是我在LeetCode上的刷题记录，主要是为了记录自己的刷题过程，以及一些解题思路，方便以后复习。

项目主要使用 go 语言为主，后续可能会使用其他语言。项目基于 go 1.21 版本，使用 go mod 管理依赖。

## 目录结构

``` bash 
    ├─definition  // LeetCode 中的数据结构定义
    ├─doc       // 一些文档
    ├─everyDay  // 每日一题
    │  └─deleteDuplicates
    ├─LCR       // 119 经典题变种挑战
    │  ├─LCR008
    │  ├─LCR009
    │  ├─LCR010
    │  └─LCR011
    └─utils     // 工具类

```

## 说明

每个题目的解题思路都会在代码中注释，如果有错误的地方，欢迎指正。
